# Trip Cost Estimator


1. Generate a itinerary for a trip using Gemini
2. Figure out the specific items that need to be purchased. This is done using gemini because of its great search capability.
3. Get the list of things to be purchased and the URL to purchase them. 
4. Use OPENAGI's computer use functionality to iterate through the list of urls and figure out the purchase price for each.
5. Summarize the things to purchase, including the total cost of the trip
